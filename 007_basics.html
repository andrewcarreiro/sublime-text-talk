SUBLIME TEXT BASICS!

  1. Finding
    1. ⌘+F Find
    2. ⌘+ALT+F Find
    3. ⌘+SHIFT+F Find & Replace in Folders
    4. Folder search inclusion & exclusion filters in "Where" field: *.txt,-*.rtf
  2. ⌘+P Goto Anything
    1. Search a file by starting with #
    2. Goto a line by starting with :
    3. Goto a function by starting with @
    4. And these can be done from outside of the file!
  3. Multi select
    1. Selection by next occurrence: ⌘+D
    2. Adding more cursors up or down: CTRL+SHIFT+UP / CTRL+SHIFT+DOWN
    3. ⌘+Left click to add a cursor in an arbitrary location
    4. ⌘+SHIFT+A, ⌘+SHIFT+L: Adds cursors on EVERY LINE
  4. Command pane
    1. Case Conversion: Uppercase (⌘+K+U), lowercase (⌘+K+L), title case, swap case
    2. Line sorting, line shuffling, deleting repeated lines (permute unique), reversing the order of lines
  5. Etcetera
    1. Bookmarks: Set up zones in a current file to jump between common sections
    2. Full screen - Do it. You're missing out otherwise
    3. Distraction Free Mode - This is how programmers code in mountainside monasteries
    4. Code Folding: Selection, ⌘+K, ⌘+level of indent
